#### 还原大师

```
我们得到了一串神秘字符串：TASC?O3RJMV?WDJKX?ZM,问号部分是未知大写字母，为了确定这个神秘字符串，我们通过了其他途径获得了这个字串的32位MD5码。但是我们获得它的32位MD5码也是残缺不全，E903???4DAB????08?????51?80??8A?,请猜出神秘字符串的原本模样，并且提交这个字串的32位MD5码作为答案。 注意：得到的 flag 请包上 flag{} 提交
```

```python
# -*- coding: utf-8 -*-
#!/usr/bin/env python
import hashlib

#print hashlib.md5(s).hexdigest().upper()
k = 'TASC?O3RJMV?WDJKX?ZM'                    #要还原的明文
for i in range(26):
	temp1 = k.replace('?',str(chr(65+i)),1)
	for j in range(26):
		temp2 = temp1.replace('?',chr(65+j),1)
		for n in range(26):
			temp3 = temp2.replace('?',chr(65+n),1)
			s = hashlib.md5(temp3.encode('utf8')).hexdigest().upper()#注意大小写
			if s[:4] == 'E903':    #检查元素
				print (s)       #输出密文
				
###
这段代码是一个Python脚本，旨在通过将给定字符串中的问号（'?'）替换为不同的字符来生成特定的MD5哈希值。目标是找到一个组合，使得MD5哈希值以前缀'E903'开头。

以下是代码的解释：

1. 导入`hashlib`模块以提供MD5哈希功能。

2. 字符串`k`表示您要恢复的目标明文。它包含问号（'?'）作为未知字符的占位符。

3. 代码使用嵌套循环遍历所有可能的字符组合来替换问号。循环使用`range()`函数并使用参数26以覆盖英文字母的所有大写字母（A-Z）。

4. 在嵌套循环内部，代码根据当前循环迭代替换第一个问号的特定字符。替换是顺序进行的，所以第一个循环替换第一个问号，第二个循环替换第二个问号，依此类推。

5. 在每次替换之后，使用`hashlib.md5()`函数对生成的字符串进行哈希。在哈希之前，使用`encode('utf8')`方法将字符串转换为字节。然后，将得到的MD5哈希转换为大写，使用`hexdigest().upper()`方法。

6. 代码检查MD5哈希的前四个字符是否等于'E903'。如果找到匹配，它将打印哈希值。

7. 这个过程会继续，直到尝试了所有可能的字符组合。

需要注意的是，通过这种方式暴力破解MD5哈希不是一个实际的方法，因为MD5被认为是一种弱哈希算法。此外，该代码没有考虑其他可能用于明文的字符或字符集。
```
